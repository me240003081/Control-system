# Altitude Control with Jerk Stabilization and Variable Mass

## Overview
This project designs a classical control system for a variable-mass aerospace vehicle to achieve altitude tracking and jerk stabilization.

## Objectives
- Model a variable-mass vertical aerospace system.
- Track a fixed altitude trajectory.
- Stabilize jerk using impulse-based modeling.
- Analyze closed-loop stability.

## Controller Used
- PD Controller
- Lead Compensation

## Results
- Stable altitude tracking with negligible steady-state error.
- Settling time approximately 4.2 s.
- Small overshoot and damped oscillations.
